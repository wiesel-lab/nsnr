# Normalized Signal-to-Noise Ratio for Covariance Estimation in Target Detection

Daniel Busbib, Tzvi Diskin, [Ami Wiesel](https://www.cs.huji.ac.il/~amiw/)

School of Computer Science and Engineering, The Hebrew University of Jerusalem

**[Paper (open access)](https://doi.org/10.1016/j.sigpro.2026.110820)** | **[Project page](https://wiesel-lab.github.io/nsnr/)**

Official implementation of *Normalized signal-to-noise ratio for covariance
estimation in target detection* (Signal Processing, vol. 250, 2027).

## Main results

**Theorem 1 (worst-case NSNR).** For positive definite $C$ and $\hat{C}$ with matrix ratio $Q = C\hat{C}^{-1}$, the worst-case NSNR has the closed form

$$\mathrm{NSNR}_{\min}(C,\hat{C}) = \min_{s} \frac{(s^{H}\hat{C}^{-1}s)^{2}}{(s^{H}\hat{C}^{-1}C\hat{C}^{-1}s)(s^{H}C^{-1}s)} = \frac{q_{\min} q_{\max}}{\left(\frac{q_{\min}+q_{\max}}{2}\right)^{2}},$$

where $q_{\min}$ and $q_{\max}$ are the extreme eigenvalues of $Q$.

**Theorem 2 (KL bounds the NSNR distance).**

$$d_{\mathrm{NSNR}}(C,\hat{C}) \le d_{\text{invariant KL}}(C,\hat{C}) \le d_{\mathrm{KL}}(C,\hat{C}),$$

and the left inequality holds with equality if and only if all interior eigenvalues of $Q$ equal the arithmetic mean $(q_{\min}+q_{\max})/2$ of the two extreme eigenvalues.

## Abstract

We address the Normalized Signal to Noise Ratio (NSNR) metric defined in the
seminal paper by Reed, Mallett, and Brennan on adaptive detection. NSNR is the
ratio between the SNR of a linear detector which uses an estimated noise
covariance and the SNR of a clairvoyant detector based on the exact unknown
covariance. It is not obvious how to evaluate NSNR since it is a function of
the target vector. To close this gap, we consider the NSNR associated with the
worst target. Using the Kantorovich Inequality, we provide a closed-form
solution for the worst-case NSNR. Then, we prove that the classical Gaussian
Kullback Leibler (KL) divergence bounds it. Motivated by these results, we
derive a simple variant of a classic norm based estimator by incorporating KL
in a leave-one-out cross-validation (LOOCV) framework. Numerical experiments
with different true covariances and various estimates suggest that the KL
metric is more correlated with the NSNR metric than competing norm-based
metrics and simply changing the metric in the LOOCV estimator improves KL and
NSNR performance.

## Installation

```bash
git clone https://github.com/wiesel-lab/nsnr.git
cd nsnr
pip install -r requirements.txt
```

Tested with Python 3.13 on macOS.

## Reproducing the results

Every figure in the paper corresponds to one script in `experiments/`:

| Result in paper | Command |
|---|---|
| Figure 1 (tightness of the Theorem 2 bound) | `python experiments/figure1.py` |
| Figure 2 (KL/MSE vs NSNR scatter)           | `python experiments/figure2.py` |
| Figure 3 (metrics vs probability of detection) | `python experiments/figure3.py` |
| Figure 4 (synthetic Toeplitz covariances)   | `python experiments/figure4.py` |
| Figure 5 (Pavia University hyperspectral)   | `bash data/download_data.sh && python experiments/figure5.py` |
| Figure 6 (Kronecker product covariances)    | `python experiments/figure6.py` |

Outputs are written to `results/`. All scripts use fixed random seeds.
Figures 4-6 run large Monte Carlo comparisons and can take a while.

## Data

Figures 1-4 and 6 use synthetic data generated by the scripts themselves.
Figure 5 uses the Pavia University hyperspectral dataset; download it with
`bash data/download_data.sh` (see [data/README.md](data/README.md)).

## Repository structure

```
src/           Core implementation: metrics (worst-case NSNR, KL, invariant KL,
               NMSE), estimators (LW, OAS, QIS, KL-LOOCV shrinkage - Algorithm 1)
               and data generation
experiments/   Scripts reproducing each figure in the paper
data/          Download script for the Pavia University dataset
docs/          Project page (served via GitHub Pages)
```
