# Data

- **Figures 1-4 and 6** use synthetic data generated by the experiment
  scripts themselves (fixed seeds); nothing to download.
- **Figure 5** uses the [Pavia University hyperspectral dataset](https://www.ehu.eus/ccwintco/index.php/Hyperspectral_Remote_Sensing_Scenes)
  (Trees class as background). Download it with:

  ```bash
  bash data/download_data.sh
  ```

  This fetches `PaviaU.mat` and `PaviaU_gt.mat` (~34 MB) into this
  directory. The `.mat` files are git-ignored and are not committed.
